## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | BerkaAnalytics Dashboard |
| **타입** | WPF 데스크톱 애플리케이션 |
| **목적** | 실제 은행 거래 데이터(Berka Dataset)를 시각화하고 이상거래를 탐지하는 분석 도구 |
| **주요 기능** | 계좌별 소비 패턴 4분할 시각화 / 이상거래 자동 탐지 / 기간 필터링 |
| **타겟 사용자** | 금융 데이터 분석가 / 은행권 IT 담당자 |

## 5. 화면별 기획서

### S-01. 시작 화면 (DB 연결)

**목적**: 앱 최초 실행 시 MySQL DB 연결 정보를 입력하고 연결을 확인한다.

**레이아웃**

```
┌─────────────────────────────────────┐
│  [로고 + 앱명]                       │
│                                     │
│  DB 연결 설정                        │
│  ┌───────────────────────────────┐  │
│  │ Host      [localhost        ] │  │
│  │ Port      [3306             ] │  │
│  │ Database  [berka            ] │  │
│  │ Username  [                 ] │  │
│  │ Password  [●●●●●●●●         ] │  │
│  └───────────────────────────────┘  │
│                                     │
│  [  연결 테스트  ]  [  시작하기  ]   │
│  ● 연결 성공 / ✕ 연결 실패 메시지   │
└─────────────────────────────────────┘
```

**주요 액션**

- 연결 테스트: DB 연결 가능 여부 확인 → 성공/실패 메시지 표시
- 시작하기: 연결 성공 상태에서만 활성화 → 메인 대시보드로 이동

---

### S-02. 메인 대시보드

**목적**: 선택한 계좌의 소비 패턴을 4분할 패널로 동시에 시각화한다.

**레이아웃**

```
┌──────────────────────────────────────────────────────────┐
│  [로고]  BerkaAnalytics    [탐지실행] [설정] [도움말]     │  툴바
├──────────────┬───────────────────────────────────────────┤
│              │  기간: [1993-01-01] ~ [1998-12-31]  [적용]│
│  계좌 목록   ├─────────────────┬─────────────────────────┤
│  [검색창]    │  패널 1         │  패널 2                  │
│              │  일별 지출 추이  │  시간×요일 히트맵        │
│  account_001 │  [라인 그래프]  │  [히트맵 이미지]         │
│  account_002 ├─────────────────┼─────────────────────────┤
│  account_003 │  패널 3         │  패널 4                  │
│  ...         │  업종별 분포    │  월별 지출 비교           │
│  [총 45개]   │  [막대 그래프]  │  [막대 그래프]           │
├──────────────┴─────────────────┴─────────────────────────┤
│  DB: 연결됨  │  계좌: account_001  │  데이터: 1,203건      │  상태바
└──────────────────────────────────────────────────────────┘
```

**주요 액션**

- 계좌 선택: 좌측 목록에서 계좌 클릭 → 4개 패널 동시 갱신
- 기간 필터: 시작일/종료일 입력 후 적용 → 전체 패널 데이터 갱신
- 탐지 실행: 현재 선택 계좌에 이상 탐지 실행 → S-03으로 전환
- 패널 커서 동기화: 패널 1 위에서 마우스 이동 시 모든 패널 커서 동기화

---

### S-02-B. 패널 1 — 일별 지출 추이

- 종류: 라인 그래프 (ScottPlot)
- X축: 날짜 / Y축: 일별 총 지출액 (CZK)
- 오버레이: 이상 탐지 구간 빨간 음영 표시
- 인터랙션: 마우스 호버 시 날짜 + 금액 툴팁

### S-02-C. 패널 2 — 시간×요일 히트맵

- 종류: 히트맵 (SkiaSharp)
- X축: 시간 (0시~23시) / Y축: 요일 (월~일)
- 색상: 거래 빈도가 높을수록 진한 색

### S-02-D. 패널 3 — 업종별 지출 분포

- 종류: 가로 막대 그래프 (ScottPlot)
- X축: 지출 금액 / Y축: 업종 코드
- 강조: 평소 없던 업종 = 빨간 테두리 표시

### S-02-E. 패널 4 — 월별 지출 비교

- 종류: 세로 막대 그래프 (ScottPlot)
- X축: 월 / Y축: 월별 총 지출액 (CZK)
- 추가: 전월 대비 증감률 텍스트 표시 (+12.3% / -5.1%)

## 6. 공통 컴포넌트

### 툴바 (상단 고정)

| 요소 | 설명 |
|------|------|
| 로고 + 앱명 | 좌측 고정 |
| 탐지 실행 버튼 | 현재 계좌 이상 탐지 실행 |
| 설정 아이콘 | S-04 설정 화면 열기 |
| 도움말 아이콘 | 사용법 툴팁 |

### 상태바 (하단 고정)

| 요소 | 내용 |
|------|------|
| DB 연결 상태 | 연결됨 / 연결 끊김 |
| 선택 계좌 | account_001 |
| 데이터 건수 | 1,203건 |
| 마지막 갱신 | 14:32:05 |

### 패널 공통 요소

| 요소 | 설명 |
|------|------|
| 패널 타이틀 | 좌상단 텍스트 |
| 로딩 인디케이터 | DB 쿼리 중 스피너 |
| 데이터 없음 상태 | "데이터가 없습니다" 중앙 표시 |
| 커서 동기화 선 | 4패널 공통 세로 점선 커서 |
