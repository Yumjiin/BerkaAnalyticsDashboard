## 1. 서비스 개요

|항목|내용|
|---|---|
|**서비스명**|BerkaAnalytics Dashboard|
|**타입**|WPF 데스크톱 애플리케이션|
|**목적**|실제 은행 거래 데이터(Berka Dataset)를 시각화하고 이상거래 탐지 결과를 분석하는 도구|
|**주요 기능**|계좌별 소비 패턴 4분할 시각화 / 이상거래 탐지 결과 조회 / 기간 필터링|

│  │           │  │  │ 일별      │ 모델별        │   │  │
│  │ - 계좌목록 │  │  │ 지출 추이 │ 이상 탐지     │   │  │
│  │ - 기간필터 │  │  │           │ 비교         │   │  │
│  │ - 검색     │  │  ├───────────┼─────────────┤   │  │

│                  │  └───────────┴─────────────┘   │  │
│                  └────────────────────────────────┘  │
└───────┬─────────────────────┬────────────────────────┘
        │                     │
        ▼                     ▼
┌──────────────┐      ┌──────────────────────┐
│ S-03.        │      │ S-04.                │
│ 이상 탐지     │      │ 설정 화면            │
│ 결과         │       │                     │
│ - 모델 카드  │       │ - DB 연결 정보 확인  │

**화면 전환 방식**

|전환|트리거|
|---|---|
|S-01 → S-02|연결 성공 후 [시작하기] 버튼 클릭|
|S-02 → S-03|계좌 선택 후 [이상 탐지 결과 보기] 버튼 클릭|
|S-02 → S-04|상단 툴바 [설정] 버튼 클릭|
|S-03 → 닫기|[X 닫기] 버튼|
|S-04 → S-01|[연결 끊기] 버튼 → StartupWindow로 이동|
|S-04 → 닫기|[닫기] 버튼|

## 4. 화면별 기획서

### S-01. 시작 화면 (DB 연결)

**목적**: 앱 최초 실행 시 MySQL DB 연결 정보를 입력하고 연결을 확인한다.

**레이아웃**

```
┌─────────────────────────────────────┐
│  [로고 + 앱명]                       │
│                                     │
│  DB 연결 설정                        │
│  ┌───────────────────────────────┐  │
│  │ Host      [localhost        ] │  │
│  │ Port      [3306             ] │  │
│  │ Database  [berka            ] │  │
│  │ Username  [                 ] │  │
│  │ Password  [●●●●●●●●         ] │  │
│  └───────────────────────────────┘  │
│                                     │
│  [  연결 테스트  ]  [  시작하기  ]   │
│  ● 연결 성공 / ✕ 연결 실패 메시지    │
└─────────────────────────────────────┘
```

**주요 액션**

|액션|동작|
|---|---|
|연결 테스트|DB 연결 가능 여부 확인 → 성공/실패 메시지 표시|
|시작하기|연결 성공 상태에서만 활성화 → 메인 대시보드로 이동|

---

### S-02. 메인 대시보드

**목적**: 선택한 계좌의 소비 패턴을 4분할 패널로 동시에 시각화한다.

**레이아웃**

```
┌──────────────────────────────────────────────────────────┐
│  [로고]  BerkaAnalytics      [이상 탐지 결과 보기] [설정]  │  ← 툴바
├──────────────┬───────────────────────────────────────────┤
│              │  기간: [1993-01-01] ~ [1998-12-31]  [적용]│
│  계좌 목록   ├─────────────────┬─────────────────────────┤
│  [검색창]    │  패널 1         │  패널 2                  │
│              │  일별 지출 추이  │  모델별 이상 탐지 비교    │
│  account_001 │  [라인 그래프]  │  [가로 막대 그래프]       │
│  account_002 ├─────────────────┼─────────────────────────┤
│  account_003 │  패널 3         │  패널 4                  │
│  ...         │  업종별 분포    │  월별 지출 비교           │
│  [총 4500개] │  [막대 그래프]  │  [막대 그래프]            │
├──────────────┴─────────────────┴─────────────────────────┤
│  DB: 연결됨  │  계좌: Account 1  │  데이터: 132일          │  ← 상태바
└──────────────────────────────────────────────────────────┘
```

**주요 액션**

| 액션                 | 동작                                                     |
| -----------          | ----------------------------------------                 |
| 계좌 선택             | 좌측 목록에서 계좌 클릭 → 4개 패널 동시 갱신                |
| 계좌 검색             | 검색창 입력 → AccountId 기준 실시간 필터링                 |
| 기간 필터             | DatePicker로 시작일/종료일 선택 후 적용 → 패널 1, 4 갱신    |
| 이상 탐지 결과 보기    | 계좌 선택 후 활성화 → S-03 창 열기                         |
| 설정                  | S-04 설정 창 열기                                         |

---

### S-02-B. 패널 1 - 일별 지출 추이

|항목|내용|
|---|---|
|차트 종류|라인 그래프 (ScottPlot 5)|
|X축|날짜|
|Y축|일별 총 지출액 (CZK)|
|오버레이|이상 탐지 날짜 빨간 수직선 표시|

---

### S-02-C. 패널 2 - 모델별 이상 탐지 비교

|항목|내용|
|---|---|
|차트 종류|가로 막대 그래프 (ScottPlot 5)|
|항목|Z-Score / Isolation Forest / Autoencoder / 고신뢰 (2개+)|
|색상|고신뢰 항목 빨간색 강조, 나머지 파란색|
|데이터|계좌별 각 모델 탐지 건수|

---

### S-02-D. 패널 3 - 업종별 지출 분포

|항목|내용|
|---|---|
|차트 종류|가로 막대 그래프 (ScottPlot 5)|
|X축|지출 금액 (CZK)|
|Y축|업종 코드 (k_symbol)|
|정렬|지출액 오름차순|

---

### S-02-E. 패널 4 - 월별 지출 비교

|항목|내용|
|---|---|
|차트 종류|세로 막대 그래프 (ScottPlot 5)|
|X축|월 (YYYY-MM)|
|Y축|월별 총 지출액 (CZK)|
|색상|전월 대비 증가 파란색 / 감소 빨간색|

### S-03. 이상 탐지 결과 화면

**목적**: 3개 모델의 탐지 건수를 카드로 비교하고, 고신뢰 이상 거래 상세 내역을 테이블로 표시한다.

**레이아웃**

```
┌──────────────────────────────────────────────────────────┐
│  이상 탐지 결과 — Account 1 · 1993-01-01 ~ 1998-12-31     │
├──────────────────────────────────────────────────────────┤
│  ┌───────────────┬───────────────┬───────────────┐       │
│  │ Z-Score       │ Isolation     │ Autoencoder   │       │
│  │ 통계 기반     │ Forest ML 기반 │ 딥러닝 기반   │       │
│  │               │               │               │       │
│  │ 4건           │ 7건           │ 3건           │       │
│  │ 임계값 z=2.5  │ 오염도 5%      │ 오차 자동산출  │       │
│  └───────────────┴───────────────┴───────────────┘       │
│                                                          │
│  고신뢰 이상 거래 (2개 이상 모델 공통 탐지) — 2건           │
│  ┌──────────────────────────────────────────────────┐    │
│  │ 날짜       금액(CZK)  Z-Score  IForest  AE오차    │    │
│  │ 1997-06-14  42,000    8.98    -0.1187  0.000000  │    │
│  │ 1997-01-12  15,300    2.58    -0.0289  0.000000  │    │
│  └──────────────────────────────────────────────────┘    │
│                                              [X 닫기]    │
└──────────────────────────────────────────────────────────┘
```

**주요 액션**

|액션|동작|
|---|---|
|X 닫기|S-03 창 닫기|

### S-04. 설정 화면

**목적**: 현재 DB 연결 정보와 이상 탐지 임계값을 확인한다. 연결 끊기로 재연결이 가능하다.

**레이아웃**

```
┌──────────────────────────────────────┐
│  설정                      [X 닫기]   │
├──────────────────────────────────────┤
│  DB 연결 정보 (읽기 전용)             │
│  Host      [127.0.0.1    ] (고정)    │
│  Port      [3306         ] (고정)    │
│  Database  [berka        ] (고정)    │
│  Username  [berka_user   ] (고정)    │
│                                      │
│  [연결 끊기] → StartupWindow로 이동   │
├──────────────────────────────────────┤
│  이상 탐지 설정 (읽기 전용)            │
│  Z-Score 임계값       2.5             │
│  Isolation Forest     0.05 (5%)      │
│  Autoencoder 오차     자동 산출       │
│                                      │
│  ※ 임계값 변경은 BerkaETLPipeline    │
│    config.py 수정 후 재실행 필요      │
├──────────────────────────────────────┤
│                          [닫기]      │
└──────────────────────────────────────┘
```

**주요 액션**

|액션|동작|
|---|---|
|연결 끊기|StartupWindow 열기 + MainWindow 닫기|
|닫기|설정 창 닫기|

## 5. 공통 컴포넌트

### 툴바 (상단 고정)

|요소|설명|
|---|---|
|로고 + 앱명|좌측 고정|
|이상 탐지 결과 보기 버튼|계좌 선택 시 활성화 → S-03 창 열기|
|설정 버튼|S-04 설정 화면 열기|

### 상태바 (하단 고정)

| 요소        | 내용                          |
| --------    | ----------------              |
| DB 연결 상태 | 연결됨 / 오류 메시지           |
| 선택 계좌    | 선택된 계좌 ID 표시            |
| 데이터 건수   | 조회된 일수 동적 표시          |
| 마지막 갱신   | 마지막 렌더링 시각 동적 표시    |

### 패널 공통 요소

|요소|설명|
|---|---|
|패널 타이틀|좌상단 텍스트 + 차트 유형 설명|
|데이터 없음 상태|빈 차트 표시|
|AutoScale|데이터 범위에 맞게 Y축 자동 조정|
